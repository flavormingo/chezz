## Your move

- **Play a friend across the world**, or across the table
- **Blitz the clock down to the wire**, or trade moves over days
- **Become a chess master** with unlimited detailed game reviews
- **Blunder in style** with an ever expanding selection of customization features

> [!NOTE]
> No account, no internet, no problem. Single player and the full game review run entirely on your phone.

## Set up the board

You'll need a Mac with Xcode and [Xcodegen](https://github.com/yonaskolb/XcodeGen).

Generate the project, fetch the engine and open Xcode with

```bash
cd ios
make bootstrap
```

Friends and online games use `server/`

Everything else just needs a phone.
